# Project Title

Uploads a public key to server/s

## Getting Started

```
git clone https://github.com/kiminzajnr/Python_Projects.git
```

```
cd learner-pub-key-uploader
python3 -m venv .venv
. .venv/bin/activate
pip install Flask
```

```
python3 app.py
```
```
touch ~/.ssh/priv.key # add your private key here
chmod 400 ~/.ssh/priv.key
```
